ntwrk.me (January 2011)

I worked on ntwrk.me for the 6.470 Web Programming Competition with a friend of mine during Junior year. The general concept was that people have different profiles in different settings and that we only want to show certain parts of ourselves to certain people. In the context of the "jobs" theme for 6.470 that year, we angled it from the prospective of applying for different jobs. If I am applying for a consulting firm vs a software company, I'd probably have a different resume or profile. So my ntwrk.me URL might be ntwrk.me/ernie/bain that I give to the consulting company, versus ntwrk.me/ernie/google for the small software company. The user can design their page using a custom WYSYWIG editor that we designed. To be frank, we never solidified exactly what we wanted the product to be, but in the end decided to just build something cool for the competition. It was a mix of PHP/jQuery/MySQL/HTML/CSS. It's not the prettiest site, but for the limited time we had over IAP whilst doing other things, we were pretty happy with what we made, and especially the editor. We ended up getting 3rd place in the competition.

OTW: One Touch Wonder (January 2010)

OTW was a project my friend and I worked on (same one as ntwrk.me) for the 6.190 Web Based Speech Recognition Competition over IAP. The task was to create any application using the speech capabilities of the WAMI toolkit developed at MIT's CSAIL. What we came up with was not one specific application, but more of a developer friendly framework around WAMI's speech capabilities. OTW was a bookmarklet, that when clicked, popped up a window that accepted speech input. We designed it so it could be fully extensible. Any developer with the APIs can plug in their own application into OTW. Some examples we gave were MBTA schedules. For instance, you could click OTW on your bookmarks bar, say "next 5 trains to Kendall station" and then the times would pop up in the window. We also showed a weather app as another example. Unfortunately, the code is no longer online, but fortunately, we won first place!
